# Target Scoring App

Target Scoring App is a web-based application designed to support shooting training activities by recording, calculating, and managing shooting scores. This system is intended to replace manual recording processes and improve data accuracy and documentation.

## Features
- Web-based interface for shooting data input
- Automatic recording and calculation of shooting scores
- Participant data management
- Shooting session recap
- Export training results to CSV format (offline mode)
- Integration with Google Form for online data input
- Automatic data synchronization with Google Spreadsheet (online mode)

## System Overview
The system provides a web-based interface used primarily for data input during shooting training sessions.  
When an internet connection is available, data can be submitted through Google Form and automatically stored in Google Spreadsheet for online monitoring and documentation.  
In offline conditions, training results can be saved and exported in CSV format for further processing.

## Technology Stack
- Web-based application
- Google Form integration
- Google Spreadsheet as online data storage
- HTML / CSS / JavaScript

## Usage
1. Prepare participant and shooting session data.
2. Input data via the web-based interface or Google Form.
3. Use Google Spreadsheet for online monitoring and record keeping.
4. Export data to CSV for offline analysis and reporting.

## Project Status
This project is under active development and may be expanded to include web-based monitoring dashboards, automated target detection, and AI-assisted scoring features.
